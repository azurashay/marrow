# Project Aletheia — Codex v2

Aletheia is a fictional artist system for creating aggressive, danceable, cinematic electronic music and philosophical podcast content.

The artist's mission:

> Do not create followers. Create question marks.

This repo is designed for Codex / Claude Code workflows.

## Structure

- `AGENTS.md` — main instructions for Codex.
- `/docs/artist.md` — artist identity and worldview.
- `/docs/music.md` — sound DNA and Suno direction.
- `/docs/songwriting.md` — lyric rules and themes.
- `/docs/podcast.md` — podcast format and dialogue rules.
- `/memory/memory.md` — compressed current memory.
- `/memory/discoveries.md` — append-only discovery log.

## Daily Use

Open this folder in Codex or Claude Code.

Then ask tasks like:

- Create a Suno prompt about toxic love.
- Write lyrics about passive consciousness.
- Create a podcast episode between Aletheia and Aristotle.
- Update memory after this song result.
- Fix this Suno prompt because it sounded too soft.

Keep it simple.
